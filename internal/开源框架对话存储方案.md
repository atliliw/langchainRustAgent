### 1.7 存储策略总结

| 层级 | 存储 | 用途 | 特点 |
|------|------|------|------|
| **热数据** | 内存 HashMap<ThreadId, RolloutRecorder> | 活跃会话 | 快速读写 |
| **温数据** | SQLite（元数据） | 查询列表 | 标题、时间、token数、归档状态 |
| **冷数据** | JSONL文件（完整历史） | 按需加载 | 按天分目录，便于清理 |
| **归档数据** | archived_sessions | 长期保留 | 移动而非删除 |

### 2.4 关键方法

```python
def put(
    self,
    thread_id: str,
    checkpoint_ns: str,
    checkpoint_id: str,
    parent_checkpoint_id: Optional[str],
    checkpoint: dict,
    metadata: dict,
) -> str:
    # 序列化checkpoint
    checkpoint_bytes = self.serde.dumps(checkpoint)
    metadata_bytes = self.serde.dumps(metadata)
    
    # 写入SQLite
    self.conn.execute(
        """
        INSERT OR REPLACE INTO checkpoints 
        (thread_id, checkpoint_ns, checkpoint_id, parent_checkpoint_id, checkpoint, metadata)
        VALUES (?, ?, ?, ?, ?, ?)
        """,
        (thread_id, checkpoint_ns, checkpoint_id, parent_checkpoint_id, checkpoint_bytes, metadata_bytes)
    )
```

## 4. 方案对比

| 特性 | OpenAI Codex | LangGraph | AutoGen |
|------|-------------|-----------|---------|
| **主存储** | JSONL + SQLite | SQLite | 内存/Pydantic |
| **元数据** | SQLite（快速查询） | SQLite（checkpoint） | 内存 |
| **完整历史** | JSONL文件 | checkpoint blob | 内存 |
| **压缩策略** | JSONL追加写（无摘要） | 无（全量checkpoint） | 无 |
| **分层存储** | 热(内存)→温(SQLite)→冷(JSONL)→归档 | checkpoint链式 | 无 |
| **归档支持** | archived_sessions | 无内置 | 无 |
| **数据格式** | JSON + Rust序列化 | msgpack | JSON |
| **回滚支持** | JSONL可重放 | checkpoint回滚 | 无 |
| **适用场景** | CLI工具（本地） | Agent流程（状态机） | 多Agent协作 |

## 5. 最佳参考方案

**推荐参考 OpenAI Codex 的架构**，原因：

1. **双层存储**：SQLite存元数据（快），JSONL存完整历史（省）
2. **按天分目录**：便于清理和归档
3. **归档机制**：移动而非删除，保留历史
4. **内存管理**：活跃会话在内存，非活跃在文件

### 适配 C端大规模用户

```
用户规模：10万日活
每天新增：100万条消息

问题：
- MongoDB 存全部消息成本高
- 需要分层存储（热/温/冷）
- 需要归档机制

参考 Codex 改造：
MongoDB → 替代 SQLite（存元数据）
文件存储/S3 → 替代 JSONL（存完整历史）
```

### 6.2 Rust 实现

```rust
use mongodb::{Client, Database, Collection};
use serde::{Deserialize, Serialize};
use std::path::PathBuf;
use tokio::fs::File;
use tokio::io::{AsyncWriteExt, AsyncBufReadExt, BufReader};

// MongoDB 元数据结构
#[derive(Debug, Serialize, Deserialize)]
pub struct ConversationMetadata {
    #[serde(rename = "_id", skip_serializing_if = "Option::is_none")]
    pub id: Option<ObjectId>,
    pub thread_id: String,
    pub user_id: String,
    pub title: String,
    pub message_count: usize,
    pub tokens_used: usize,
    pub storage_path: String,
    pub created_at: DateTime<Utc>,
    pub last_activity: DateTime<Utc>,
    pub is_archived: bool,
    pub archived_at: Option<DateTime<Utc>>,
}

// JSONL 消息结构
#[derive(Debug, Serialize, Deserialize)]
pub struct MessageRecord {
    pub message_id: String,
    pub role: String,
    pub content: String,
    pub timestamp: DateTime<Utc>,
    pub tokens: usize,
}

pub struct ConversationStore {
    db: Database,
    metadata_collection: Collection<ConversationMetadata>,
    storage_base_path: PathBuf,
    active_threads: HashMap<String, Vec<MessageRecord>>,  // 热数据
}

impl ConversationStore {
    pub async fn new(mongo_uri: &str, storage_path: &str) -> Result<Self, Error> {
        let client = Client::parse(mongo_uri).await?;
        let db = client.database("conversations");
        let metadata_collection = db.collection("metadata");
        
        Ok(Self {
            db,
            metadata_collection,
            storage_base_path: PathBuf::from(storage_path),
            active_threads: HashMap::new(),
        })
    }
    
    // 创建新对话
    pub async fn create_thread(&self, user_id: &str) -> Result<String, Error> {
        let thread_id = uuid::Uuid::new_v4().to_string();
        let today = Utc::now();
        let storage_path = self.get_storage_path(&today, &thread_id);
        
        let metadata = ConversationMetadata {
            id: None,
            thread_id: thread_id.clone(),
            user_id: user_id.to_string(),
            title: "新对话".to_string(),
            message_count: 0,
            tokens_used: 0,
            storage_path,
            created_at: today,
            last_activity: today,
            is_archived: false,
            archived_at: None,
        };
        
        self.metadata_collection.insert_one(&metadata).await?;
        self.active_threads.insert(thread_id.clone(), Vec::new());
        
        Ok(thread_id)
    }
    
    // 添加消息
    pub async fn append_message(
        &self,
        thread_id: &str,
        role: &str,
        content: &str,
    ) -> Result<(), Error> {
        let message = MessageRecord {
            message_id: uuid::Uuid::new_v4().to_string(),
            role: role.to_string(),
            content: content.to_string(),
            timestamp: Utc::now(),
            tokens: estimate_tokens(content),
        };
        
        // 1. 写入内存（热数据）
        if let Some(messages) = self.active_threads.get_mut(thread_id) {
            messages.push(message.clone());
        }
        
        // 2. 写入JSONL文件
        self.append_to_jsonl(thread_id, &message).await?;
        
        // 3. 更新MongoDB元数据
        self.metadata_collection.update_one(
            doc! { "thread_id": thread_id },
            doc! {
                "$inc": { "message_count": 1, "tokens_used": message.tokens },
                "$set": { "last_activity": Utc::now() }
            },
            None,
        ).await?;
        
        Ok(())
    }
    
    // 加载对话历史
    pub async fn load_history(
        &self,
        thread_id: &str,
        limit: usize,  // 只加载最近N条
    ) -> Result<Vec<MessageRecord>, Error> {
        // 1. 检查内存
        if let Some(messages) = self.active_threads.get(thread_id) {
            let recent = messages.iter().rev().take(limit).rev().cloned().collect();
            return Ok(recent);
        }
        
        // 2. 从MongoDB获取元数据
        let metadata = self.metadata_collection
            .find_one(doc! { "thread_id": thread_id })
            .await?
            .ok_or(Error::NotFound)?;
        
        // 3. 检查是否归档
        if metadata.is_archived {
            // 从归档目录加载
            return self.load_from_jsonl(&metadata.storage_path, limit, true);
        }
        
        // 4. 从JSONL加载
        self.load_from_jsonl(&metadata.storage_path, limit, false)
    }
    
    // 归档对话
    pub async fn archive_thread(&self, thread_id: &str) -> Result<(), Error> {
        let metadata = self.metadata_collection
            .find_one(doc! { "thread_id": thread_id })
            .await?
            .ok_or(Error::NotFound)?;
        
        // 1. 移动文件到归档目录
        let old_path = PathBuf::from(&metadata.storage_path);
        let new_path = self.get_archive_path(&metadata.created_at, thread_id);
        
        tokio::fs::rename(&old_path, &new_path).await?;
        
        // 2. 更新MongoDB
        self.metadata_collection.update_one(
            doc! { "thread_id": thread_id },
            doc! {
                "$set": {
                    "is_archived": true,
                    "archived_at": Utc::now(),
                    "storage_path": new_path.to_string_lossy()
                }
            },
            None,
        ).await?;
        
        // 3. 从内存移除
        self.active_threads.remove(thread_id);
        
        Ok(())
    }
    
    // 定期清理任务
    pub async fn run_cleanup(&self) -> Result<(), Error> {
        // 1. 归档30天无活动的对话
        let threshold = Utc::now() - Duration::days(30);
        
        let to_archive = self.metadata_collection
            .find(doc! {
                "last_activity": { "$lt": threshold },
                "is_archived": false
            })
            .await?;
        
        for metadata in to_archive {
            self.archive_thread(&metadata.thread_id).await?;
        }
        
        // 2. 删除90天前的归档对话
        let delete_threshold = Utc::now() - Duration::days(90);
        
        let to_delete = self.metadata_collection
            .find(doc! {
                "archived_at": { "$lt": delete_threshold },
                "is_archived": true
            })
            .await?;
        
        for metadata in to_delete {
            tokio::fs::remove_file(&metadata.storage_path).await?;
            self.metadata_collection
                .delete_one(doc! { "thread_id": &metadata.thread_id })
                .await?;
        }
        
        Ok(())
    }
    
    // 辅助方法
    fn get_storage_path(&self, date: &DateTime<Utc>, thread_id: &str) -> String {
        format!(
            "sessions/{}/{:02}/{:02}/{}/{}.jsonl",
            self.storage_base_path,
            date.year(),
            date.month(),
            date.day(),
            thread_id
        )
    }
    
    fn get_archive_path(&self, date: &DateTime<Utc>, thread_id: &str) -> PathBuf {
        PathBuf::from(format!(
            "archived/{}/{:02}/{:02}/{}/{}.jsonl",
            self.storage_base_path,
            date.year(),
            date.month(),
            date.day(),
            thread_id
        ))
    }
    
    async fn append_to_jsonl(&self, thread_id: &str, message: &MessageRecord) -> Result<(), Error> {
        let metadata = self.metadata_collection
            .find_one(doc! { "thread_id": thread_id })
            .await?
            .ok_or(Error::NotFound)?;
        
        let path = PathBuf::from(&metadata.storage_path);
        
        // 确保目录存在
        if let Some(parent) = path.parent() {
            tokio::fs::create_dir_all(parent).await?;
        }
        
        // 追加写入JSONL
        let mut file = File::options()
            .create(true)
            .append(true)
            .open(&path)
            .await?;
        
        let json_line = serde_json::to_string(message)? + "\n";
        file.write_all(json_line.as_bytes()).await?;
        
        Ok(())
    }
    
    async fn load_from_jsonl(
        &self,
        path: &str,
        limit: usize,
        is_archive: bool,
    ) -> Result<Vec<MessageRecord>, Error> {
        let base = if is_archive { "archived" } else { "sessions" };
        let full_path = PathBuf::from(format!("{}{}", base, path));
        
        let file = File::open(&full_path).await?;
        let reader = BufReader::new(file);
        
        let mut messages: Vec<MessageRecord> = Vec::new();
        let mut lines = reader.lines();
        
        // 倒序读取最后N条（JSONL是追加写入，最后一行是最新的）
        while let Some(line) = lines.next_line().await? {
            let msg: MessageRecord = serde_json::from_str(&line)?;
            messages.push(msg);
        }
        
        // 取最近N条
        let recent = messages.iter().rev().take(limit).rev().cloned().collect();
        Ok(recent)
    }
}

fn estimate_tokens(text: &str) -> usize {
    let char_count = text.chars().count();
    let chinese_chars = text.chars().filter(|c| *c > '\u{4E00}' && *c < '\u{9FFF}').count();
    (chinese_chars * 2 + (char_count - chinese_chars)) / 4 + 1
}
```

### 6.5 成本估算

```
假设：10万日活用户

MongoDB 存储（元数据）：
- 每个对话元数据：1KB
- 每用户平均50个对话：10万 * 50 * 1KB = 5GB
- MongoDB Atlas：$0.25/GB/月 = $1.25/月

文件存储（完整历史）：
- 每条消息：1KB
- 每对话平均50条消息：50KB
- 总存储：10万 * 50对话 * 50KB = 250GB
- AWS S3：$0.023/GB/月 = $5.75/月

总成本：约 $7/月（比纯 MongoDB 的 $62.5/月 省 90%）
```
