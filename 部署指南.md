# LangChainRust Demo 部署指南（三种搜索模式版）

## 一、功能概述

本 Demo 支持 **三种搜索模式对比测试**：

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| **向量检索** | 纯语义搜索，使用 OpenAI Embeddings | 自然语言查询、语义理解 |
| **BM25检索** | 关键词检索，基于词频统计 | 精确关键词、产品代码、ID匹配 |
| **混合检索** | RRF融合（向量 + BM25） | 综合-best of both worlds |
| **对比测试** | 同时执行三种搜索并对比结果 | 评估检索效果 |

---

## 二、API 端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/upload` | POST | 上传文件（自动存入向量+BM25） |
| `/api/search/vector` | POST | 向量检索 |
| `/api/search/bm25` | POST | BM25关键词检索 |
| `/api/search/hybrid` | POST | RRF混合检索 |
| `/api/search/compare` | POST | 三种检索对比测试 |
| `/api/stats` | GET | 统计信息（向量文档数+BM25索引数） |
| `/api/clear` | POST | 清空所有数据 |

```bash

Description=LangChainRust Demo Backend

Type=simple
User=root
```

### 步骤 11: 安装 Nginx


### 步骤 12: 配置 Nginx

```bash
cat > /etc/nginx/nginx.conf << 'EOF'
user nginx;
worker_processes auto;
error_log /var/log/nginx/error.log;
pid /run/nginx.pid;

events {
    worker_connections 1024;
}

http {
    log_format main '$remote_addr - $remote_user [$time_local] "$request" '
                    '$status $body_bytes_sent "$http_referer" '
                    '"$http_user_agent"';
    access_log /var/log/nginx/access.log main;
    sendfile on;
    tcp_nopush on;
    keepalive_timeout 65;
    include /etc/nginx/mime.types;
    default_type application/octet-stream;
    
    server {
        listen 80 default_server;
        server_name _;
        root /opt/langchainrust/demo/frontend;
        index index.html;
        
        # API 代理
        location /api/ {
            proxy_pass http://127.0.0.1:8080/api/;
            proxy_http_version 1.1;
            proxy_set_header Host $host;
            proxy_set_header X-Real-IP $remote_addr;
            proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
            client_max_body_size 50M;
        }
        
        # 前端静态文件
        location / {
            try_files $uri $uri/ /index.html;
        }
        
        # 静态资源缓存
        location ~* \.(css|js|png|jpg|jpeg|gif|ico)$ {
            expires 1d;
        }
    }
}
EOF

# 清理默认配置
rm -f /etc/nginx/conf.d/*.conf

# 验证配置
nginx -t

# 启动
systemctl restart nginx
```
