# LangChainRust Agent 部署指南

## 一、架构说明

**前后端分离架构：**

```
┌─────────────────────────────────────────────────────────┐
│                    用户浏览器                            │
│                 http://192.168.10.100:8080              │
└─────────────────────┬───────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────────┐
│                   Nginx (端口 8080)                      │
│                                                         │
│   /              → 前端静态文件                          │
│                    /opt/langchainrust/demo/frontend/     │
│                                                         │
│   /api/*         → 代理到后端                            │
│                    http://127.0.0.1:8090/api/*           │
└─────────────────────┬───────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────────┐
│              Axum 后端 (端口 8090)                       │
│                                                         │
│   只处理 /api/* 请求，不返回前端                          │
│                                                         │
│   /api/upload    → 文件上传                              │
│   /api/search    → 搜索（向量/BM25/混合）                 │
│   /api/chat      → RAG对话                               │
│   /api/stats     → 统计信息                              │
│   /api/documents → 文档管理                              │
└─────────────────────┬───────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────────┐
│                数据存储层                                │
│                                                         │
│   Qdrant (6334)   → 向量存储                             │
│   MongoDB (27017) → BM25索引                             │
│   SQLite          → 对话历史                             │
└─────────────────────────────────────────────────────────┘
```

**端口分配：**
| 服务 | 端口 | 说明 |
|------|------|------|
| Nginx | 8080 | 前端静态文件 + API反向代理 |
| 后端 | 8090 | 只处理API请求 |
| Qdrant | 6333/6334 | 向量数据库 |
| MongoDB | 27017 | BM25文档存储 |

## 四、Nginx配置

`/etc/nginx/nginx.conf`:

```nginx
user nginx;
worker_processes auto;
error_log /var/log/nginx/error.log;
pid /run/nginx.pid;

events {
    worker_connections 1024;
}

http {
    log_format main '$remote_addr - $remote_user [$time_local] "$request" '
                    '$status $body_bytes_sent "$http_referer" '
                    '"$http_user_agent"';
    access_log /var/log/nginx/access.log main;
    sendfile on;
    tcp_nopush on;
    keepalive_timeout 65;
    include /etc/nginx/mime.types;
    default_type application/octet-stream;
    
    server {
        listen 8080 default_server;
        server_name _;
        root /opt/langchainrust/demo/frontend;
        index index.html;
        
        # API反向代理
        location /api/ {
            proxy_pass http://127.0.0.1:8090/api/;
            proxy_http_version 1.1;
            proxy_set_header Host $host;
            proxy_set_header X-Real-IP $remote_addr;
            proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
            client_max_body_size 50M;
        }
        
        # 前端静态文件
        location / {
            try_files $uri $uri/ /index.html;
        }
        
        # 静态资源缓存
        location ~* \.(css|js|png|jpg|jpeg|gif|ico)$ {
            expires 1d;
        }
    }
}
```

## 五、Systemd服务

**后端服务 `/etc/systemd/system/langchainrust-agent.service`:**

```ini
[Unit]
Description=LangChainRust Agent Backend (API only)
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/opt/langchainrust/demo
ExecStart=/opt/langchainrust/demo/target/release/langchainrust-agent
Restart=always
RestartSec=5
Environment="RUST_LOG=info"
Environment="LD_LIBRARY_PATH=/usr/lib64"

[Install]
WantedBy=multi-user.target
```

## 六、部署流程

### 1. 编译本地代码

```bash
cd langchainRustAgent
cargo build --release
```

### 2. 打包后端代码

```bash
tar --exclude='target' --exclude='uploads' --exclude='frontend' --exclude='internal' --exclude='*.md' \
    -cvf backend.tar Cargo.toml Cargo.lock config.toml crates src
```

### 3. 打包前端代码

```bash
cd frontend
tar -cvf frontend.tar index.html css js
```

### 4. 上传到服务器

```bash
scp backend.tar root@192.168.10.100:/opt/langchainrust/demo/
scp frontend.tar root@192.168.10.100:/opt/langchainrust/demo/frontend/
```

### 5. 服务器解压编译

```bash
ssh root@192.168.10.100

cd /opt/langchainrust/demo
rm -rf src crates Cargo.toml Cargo.lock config.toml
tar -xf backend.tar

cd frontend
tar -xf frontend.tar

cd /opt/langchainrust/demo
source ~/.cargo/env
cargo build --release
```

### 6. 配置服务

```bash
# 配置Nginx
cat > /etc/nginx/nginx.conf << 'EOF'
# （见上文Nginx配置）
EOF

# 配置后端服务
cat > /etc/systemd/system/langchainrust-agent.service << 'EOF'
# （见上文Systemd配置）
EOF

# 清理默认配置
rm -f /etc/nginx/conf.d/*.conf

# 启动服务
systemctl daemon-reload
systemctl enable nginx
systemctl enable langchainrust-agent
systemctl start nginx
systemctl start langchainrust-agent
```

### 7. 验证部署

```bash
# 测试前端（8080）
curl http://localhost:8080/ | head -10

# 测试API（8080通过Nginx）
curl http://localhost:8080/api/stats

# 测试后端直接访问（8090，只能访问API）
curl http://localhost:8090/api/stats
```

## 九、更新流程

### 更新前端

```bash
cd frontend && tar -cvf frontend.tar index.html css js
scp frontend.tar root@192.168.10.100:/opt/langchainrust/demo/frontend/
ssh root@192.168.10.100 "cd /opt/langchainrust/demo/frontend && tar -xf frontend.tar"
# 无需重启
```

### 更新后端

```bash
tar --exclude='target' --exclude='uploads' --exclude='frontend' \
    -cvf backend.tar Cargo.toml Cargo.lock config.toml crates src
scp backend.tar root@192.168.10.100:/opt/langchainrust/demo/
ssh root@192.168.10.100 "cd /opt/langchainrust/demo && tar -xf backend.tar && cargo build --release && systemctl restart langchainrust-agent"
```
